# Puhuva veistos

A minimalist browser-based audio interface for an interactive sculpture. The prototype uses Gemini Live for real-time voice interaction and frames the sculpture as a situated philosophical presence.

This repository is preserved as part of the Elopeli-adjacent prototype history. See [META_CONTEXT.md](META_CONTEXT.md) for the research context, findings, and lineage.

Original AI Studio app: https://ai.studio/apps/drive/1OStQLahtn1jO5r8Sx54Jzh6Tiw4YMNEQ

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env.local` and set `GEMINI_API_KEY`.
3. Run the app:
   `npm run dev`
