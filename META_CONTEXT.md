# Meta Context

## Purpose

This demo preserves the Puhuva veistos / Sentient Sculpture Audio Interface experiment from the Elopeli-adjacent prototype lineage.

The prototype explores a browser-based audio interface for an interactive sculpture. The sculpture is framed as a calm, philosophical presence that responds through Gemini Live audio and invites visitors to reflect on what the sculpture is from their own perspective.

## What It Proved

- A language model can be bound to a specific situated voice, role, and interaction context rather than used as a generic chat assistant.
- Real-time speech interaction can make an AI-mediated artifact feel like a presence in a gallery or installation setting.
- Prompt engineering can constrain the model toward subjectivity, observer relationship, and contextual speech without requiring a traditional game loop.

## Main Limitation

This is not a core Elopeli engine. It is a situated audio-interface study for a single artifact, not a multi-player LARP system with private prompts, shared game state, scene structure, or dramaturgical branching.

## Main Finding

The important design move is contextual voice-binding: the model becomes useful when it is attached to a clear role, physical situation, conversational stance, and thematic constraint.

## Lineage

- Branch: adjacent method study.
- Related Elopeli lineage: part of the wild tests after the LGM/world-state dead end.
- Conceptual connection: helped clarify how to bind LLM output to a specific topic, voice, and social situation.
- Later relevance: supports Elopeli's broader taxonomy and situated-prompt thinking, especially for embodied interfaces and role-bound AI presence.
